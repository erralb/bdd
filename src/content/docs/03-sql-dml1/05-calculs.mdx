En SQL, il est possible de spécifier des données dérivées ou des constantes :

```sql
SELECT 'TVA de', numero, '=', 0.2 * prix * stock 
FROM Produits
WHERE stock > 10 ;
```

_Dans cet exemple, on calcule la TVA pour les produits dont le stock est supérieur à 10._

## Renommage des colonnes

Par défaut, les colonnes du résultat prennent le nom utilisé dans la clause `SELECT`

Pour utiliser un autre nom (ou **alias**) il faut utiliser la clause `AS`. 

Exemple :

```sql
SELECT numero AS 'Produit', 0.2 * prix * stock AS 'TVA' 
FROM Produits 
WHERE stock > 10 ;
```
