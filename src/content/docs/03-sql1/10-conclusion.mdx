## Conclusion

* Les requêtes SQL de sélection permettent d'**extraire** des données d'une table
* Les requêtes SQL de sélection peuvent contenir des **conditions** de sélection
* Les requêtes SQL de sélection peuvent contenir des **opérateurs** de **comparaison**
* Les requêtes SQL de sélection peuvent contenir des opérateurs **logiques**
* Les requêtes SQL de sélection peuvent contenir des opérateurs de **regroupement**
* Les requêtes SQL de sélection peuvent contenir des **fonctions agrégatives**
* Les requêtes SQL de sélection peuvent contenir des **sous-requêtes**
* Les requêtes SQL de sélection peuvent contenir des **quantifieurs ensemblistes**
